In fluorender/Install/Ubuntu, create:
fluorender/
├── debian/
│   └── control
│   ├── rules
│   ├── compat
│   └── install
├── build/
│   └── bin/fluorender

Copy fluorender/build/bin to build/bin

chmod +x fluorender/debian/rules
chmod 755 fluorender/build/bin/FluoRender

cd fluorender/Install/Ubuntu/fluorender
./build_and_test.sh
