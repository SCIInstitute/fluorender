In fluorender_install, create:
fluorender/
├── DEBIAN/
│   └── control
│   ├── rules
│   ├── compat
│   └── install
├── build/   (your compiled output)

├── usr/
│   ├── bin/
│   │   └── fluorender
│   └── share/
│       ├── applications/
│       │   └── fluorender.desktop
│       └── icons/
│           └── hicolor/256x256/apps/fluorender.png

Copy fluorender/build/bin to usr/bin
Add control file to DEBIAN


chmod -R 755 fluorender/usr
chmod 644 fluorender/DEBIAN/control
chmod +x debian/rules

dpkg-deb --build fluorender
